# Channeled Backend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.example .env
```

3. Update `.env` with your configuration

## Development

Run the development server:
```bash
npm run dev
```

## Build

Compile TypeScript:
```bash
npm run build
```

## Test

Run tests:
```bash
npm test
```
